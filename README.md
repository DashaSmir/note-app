# Note-app
Этот репозиторий содержит код учебного проекта на Go и представляет из себя приложение-заметки с функциями добавления, удаления и просмотра текущих заметок.

Чтобы запустить приложение - необходимо склонировать репозиторий, и из корневой папки проекта выполнить команду
```bash
# Запуск приложения на http://localhost:8080/
go run ./cmd/web
```
